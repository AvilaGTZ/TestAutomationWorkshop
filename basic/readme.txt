# CETI Automation python course #

# Requirements --------------------------------- #
- python 3.7 installed
- google chrome installed

# Install python requirements ------------------- #
- Open a cmd window and withinthe project folder execute: 'pip install -r requirements.txt'

# Execute the selenium basic test --------------- #
- Open a cmd window within the project and execute: python 'basicTest.py'

# Execute the pytest basic test ----------------- #
- Open a cmd window within the project and execute: 'pytest -s -v'

# Execute the dummy report generator example ---- #
- Open a cmd window within the project and execute: 'python reportGenerator.py'
